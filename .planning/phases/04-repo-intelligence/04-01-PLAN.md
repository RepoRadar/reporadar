    Export `async function fetchRepoContext(fullName: string): Promise<RepoContext>`:

    5. Apply `trimReadme(rawReadme)` and `capTree(treeItems)`. License = `m.license?.spdx_id ?? null`. Return the full `RepoContext`. `treePathsTruncated = apiTruncated || capResult.truncated`.
