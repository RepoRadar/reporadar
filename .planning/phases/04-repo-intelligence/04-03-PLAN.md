| README / file paths → DOM | untrusted markdown and tree paths rendered in the browser |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-11 | Tampering | XSS via README markdown or assistant content | mitigate | react-markdown safe-by-default, NO rehype-raw, no dangerouslySetInnerHTML; links `target="_blank" rel="noopener noreferrer"` |
| T-04-12 | Tampering | malformed owner/repo path param | mitigate | `isValidFullName` gate before fetch; failure renders the clean not-available message, no stack trace |
| T-04-13 | Information Disclosure | raw error / stack trace shown to user | mitigate | try/catch renders the styled "This repo is not available." copy only |
